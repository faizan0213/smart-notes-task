# 📝 Flutter Notes App

A beautifully designed **Flutter Firebase Notes App** built using **Provider**, **Firebase Auth**, **Cloud Firestore**, and a clean scalable architecture.  
Easily add and manage your personal notes with smooth UX and clean code.

---

## 🚀 Features

- ✅ Add notes with title & message
- ✅ Firebase Email/Password Authentication
- ✅ Save notes in Cloud Firestore
- ✅ Clean architecture for scalability
- ✅ State management using `Provider`
- ✅ Smooth UI/UX with keyboard handling and loading
- ✅ Works well on both mobile and tablet

---

## 🔧 Tech Stack

| Tech          | Usage                            |
| ------------- | -------------------------------- |
| Flutter       | Cross-platform UI                |
| Firebase      | Backend services                 |
| Firebase Auth | User login system                |
| Firestore     | Realtime database                |
| Provider      | Robust state management          |
| Clean Code    | Scalable & readable architecture |

---

## 🧠 State Management: Provider

### 🔐 AuthProvider

Handles:

- Login
- Logout
- Current user session

### 📄 FirestoreProvider

Handles:

- Add new notes
- Stream user-specific notes
